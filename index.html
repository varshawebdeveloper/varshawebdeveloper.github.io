CASE_01 //

Bespoke Editorial Portfolio

HTML5 CSS3 JavaScript Web3Forms

My Role

UI Designer & Frontend Developer

The Challenge

Deliver a fast, responsive, and preflight-free contact form that bypasses strict preflight CORS restrictions (`OPTIONS` requests) on mobile carriers.

The Approach

Designed an asymmetrical minimal layout. Refactored the form submit fetch handler to send raw browser `FormData` directly, converting the API call into a standard "Simple Request" that avoids CORS preflight checks.

The Outcome

A responsive single-page site with instant, preflight-free form delivery verified across Android, iOS, and desktop viewports.
